Построить машину Тьюринга, вычисляющую функцию g(m, n) 
от натуральных аргументов m и n (5 баллов).

П р и м е ч а н и е . Используемые ниже функции определяются следующим образом:

sg(m) = { 0, m = 0;
        { 1, m > 0;

12. g(m, n) = 5 + sg(|m − n|).

Решение:

"  := пробел (:= ␣)
.  := финальное состояние
[] := положение курсора

1. g₁(m, n) = |m - n|

	Пример:
	1. 011100 	-> 01110
	2. 001110	-> 01110
	3. 0110110	-> 00
	5. 0111010	-> 0110
	5. 0101110	-> 0110

	Шаги:
	// 01: ([0] 1^n 0 1^m 0)
	01 0 -> 02 " R

	// 02: ([1^n 0] 1^m 0)
	02 1 -> 02 1 R
	02 0 -> 03 0 R

	// 03: (1^n 0 [1^m 0])
	03 1 -> 03 1 R
	03 0 -> 04 " L

	// 04: (1^n [0 1^m])
	04 0 -> 05 0 L	// m == 0
	04 1 -> 06 " L	// m <> 0

	// 05: m == 0, ([" 1^n] 0)
	05 0 -> 05 0 L
	05 1 -> 05 1 L
	05 " ->  . 0 .	// .: ([0] 1^n 0)

	// 06: m <> 0, ([" 1^n 0 1^(m-1)])
	06 0 -> 06 0 L
	06 1 -> 06 1 L
	06 " -> 07 " R

	// 07: m <> 0, ([1^n 0] 1^(m-1))
	07 0 -> 08 0 R	// n == 0
	07 1 -> 11 " R	// n <> 0 -> n := n-1, m := m-1

	// 08: m <> 0, n == 0, (0 [1^(m-1)])
	08 1 -> 08 1 R
	08 " -> 09 1 R

	// 09: m <> 0, n == 0, (0 1^m ["])
	09 " -> 10 0 L

	// 10: m <> 0, n == 0, ([0 1^m] 0)
	10 1 -> 10 1 L
	10 0 ->  . 0 .	// .: ([0] 1^m 0)

	// 11: ([1^n 0] 1^m)
	11 1 -> 11 1 R
	11 0 -> 12 0 R

	// 12: (1^n 0 [1^m "])
	12 1 -> 12 1 R
	12 " -> 04 " L

2. g₂(n) = sg(n)

	Пример:
	1. 00       -> 00
	2. 010      -> 010
	3. 0110     -> 010
	4. 01110    -> 010
	5. 0111110  -> 010

	Шаги:
	// 01: ([0] 1^n 0)
	01 0 -> 02 " R

	// 02: ([1^n 0])
	02 0 -> 03 0 L	// n == 0
	02 1 -> 04 " R	// n <> 0

	// 03: n == 0, (["] 0) | n <> 0, (["] 1 0) 
	03 " ->  . 0 .	// .: ([0] 0)

	// 04: n <> 0, ([1^n 0])
	04 1 -> 04 " R
	04 0 -> 05 0 L

	// 05: n <> 0, (["] 0)
	05 " -> 03 1 L

3. g₃(n) = 5 + n

	Пример:
	1. 00       -> 0111110
	2. 010      -> 01111110
	3. 0110     -> 011111110
	4. 01110    -> 0111111110
	5. 0111110  -> 011111111110

	Шаги:
	// 01: ([0] 1^n 0)
	01 0 -> 02 1 L

	// 02: (["] 1^(n+1) 0)
	02 " -> 03 1 L

	// 03: (["] 1^(n+2) 0)
	03 " -> 04 1 L

	// 04: (["] 1^(n+3) 0)
	04 " -> 05 1 L

	// 05: (["] 1^(n+4) 0)
	05 " -> 06 1 L

	// 06: (["] 1^(n+5) 0)
	06 " ->  . 0 .	// .: ([0] 1^(n+5) 0)
